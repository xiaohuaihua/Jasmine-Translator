# Trademark and Brand Usage

The source code of Jasmine Translator / 花茶翻译器 is licensed under Apache-2.0, but the project brand is not.

Reserved brand assets include:

- The name “花茶翻译器”
- The name “Jasmine Translator”
- Project icons and logos
- Store screenshots and promotional images
- Official release identity

You may fork and modify the code under the Apache-2.0 license, but you may not publish a modified version using the same or confusingly similar name, icon, screenshots, or official identity unless you have explicit permission from the maintainer.

If you distribute a modified build, make it clear that it is unofficial and use a different name and visual identity.
