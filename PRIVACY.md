# Privacy Policy

Last updated: 2026-07-04

Jasmine Translator / 花茶翻译器 translates selected text and page text with the AI provider configured by the user.

## Local storage

The extension stores provider settings, endpoint, API key, model ID, language preferences, translation cache, and site preferences in browser extension storage.

## Translation requests

When you request translation, the selected text or page text is sent to the provider endpoint you configured. The default preset is TokenDance. Custom endpoints are also supported.

## API keys

API keys are stored locally in browser extension storage. Jasmine Translator does not provide a shared API key and does not operate a translation server.

## Data practices

Jasmine Translator does not sell user data, inject remote code, or upload API keys to a maintainer server.

Third-party AI providers process submitted text according to their own terms and privacy policies.
