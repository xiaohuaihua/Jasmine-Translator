# Changelog

## 0.6.0 - 2026-07-04

Initial public release preparation for Jasmine Translator / 花茶翻译器.

### Added

- Selected text translation
- Full page translation
- Pause and resume controls
- Progress capsule with drag support
- Hide and restore translated text
- Local translation cache
- TokenDance provider preset
- Custom OpenAI-compatible endpoint support
- Provider setup links and recommended model shortcut
- Cached model list fetching with timeout handling

### Notes

This release uses your own API key. The extension does not provide a shared translation service.
