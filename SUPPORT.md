# Support

Jasmine Translator / 花茶翻译器 is a personal open-source project.

## Support scope

Best-effort support is provided through GitHub Issues.

The maintainer does not guarantee:

- A response time
- Compatibility with every website
- Stability or pricing of third-party AI providers
- Recovery of lost API keys
- Support for unofficial modified builds

## Before opening an issue

Please include:

- Browser name and version
- Extension version
- The website where the issue happens, if it can be shared
- Provider preset or endpoint type
- A screenshot or console error, if available

Do not include your API key in issues, screenshots, logs, or videos.

## Commercial contact

For provider partnerships or commercial discussions, please contact the maintainer through GitHub.
