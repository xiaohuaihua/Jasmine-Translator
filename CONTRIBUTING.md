# Contributing

Thanks for your interest in Jasmine Translator / 花茶翻译器.

This repository is primarily a user-facing release repository. Contributions are welcome, but maintenance is best-effort.

## Development

Install dependencies and build the content script:

```bash
npm install
npm run build
```

Load the repository folder as an unpacked extension in Chrome or Edge.

## Pull requests

Please keep changes focused and explain:

- What problem the change solves
- How it was tested
- Whether it affects permissions, privacy, or provider behavior

Do not include API keys, private endpoints, generated secrets, or unrelated local files.
